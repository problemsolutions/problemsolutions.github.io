# Validate HTML & CSS

Run validation checks on the site files.

## Checks to perform:

1. **HTML Validation** - Check for syntax errors and best practices
2. **CSS Validation** - Verify stylesheet is valid
3. **Accessibility** - Quick a11y review
4. **Link Check** - Verify all links are valid

## Manual Validation
- W3C HTML: https://validator.w3.org/#validate_by_input
- W3C CSS: https://jigsaw.w3.org/css-validator/#validate_by_input

## Quick Checks
- All `<img>` tags have `alt` attributes
- All `<a>` tags have `href` or proper ARIA
- Semantic HTML structure (header, main, footer)
- Meta tags present (viewport, description, og:*)
